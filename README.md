# Company Web

مشروع Next.js بسيط لصفحة هبوط (Landing Page) بخلفية فيديو ونص في المنتصف.

## التشغيل

```bash
npm install
npm run dev
```

ثم افتح:

`http://localhost:3000`

## هيكلة المشروع الحالية

- `src/app/page.tsx` صفحة الهبوط
- `src/app/globals.css` تنسيقات الصفحة
- `public/videos/photographer-studio-cinematic.mp4` فيديو الخلفية
- `assets/fonts/libre-baskerville` ملفات الخط (للاستخدام لاحقًا)
